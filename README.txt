- icons/icon-512.png

تشغيل محلي (مستحسن):
1) افتح سطر الأوامر في مجلد Maktabat_AlAysaei_vFinal.
2) لتشغيل خادم محلي سريع:
   - إن كان لديك Node.js: نفذ `npx http-server -c-1` أو `npx http-server` ثم افتح http://localhost:8080
   - أو استخدم Live Server في VSCode.
3) افتح Chrome إلى http://localhost:8080/index.html
4) سيظهر زر "تثبيت" إن دعم المتصفح ذلك. اضغط لتثبيت التطبيق (سيعمل أوفلاين بعد التثبيت).

لوحة الضبط:
- اضغط ⚙️ لتعيين كلمة مرور المسؤول أول مرة (أدخل وكوّن كلمة مرور ثم أكدها).
- بعد ذلك سيتطلب الضبط إدخال كلمة المرور.
- يمكنك تغيير كلمة المرور من داخل الضبط أو إعادة ضبطها (تحذف وتطلب تعيينها من جديد).

استيراد/تصدير:
- استخدم أزرار استيراد/تصدير في لوحة الضبط (JSON) لنقل/نسخ الفهرس.

ملاحظة عن التثبيت:
- ميزة التثبيت PWA تعمل أفضل عندما تكون الصفحة مقدمة عبر HTTP(s) (localhost أو خادم فعلي). فتح الملف عبر file:// قد لا يظهر زر التثبيت.

إذا رغبت، سأعطيك الآن:
- أوامر سريعة لإنشاء ZIP على النظام لديك (Windows / macOS / Linux).
- أو أجهّز لك ملف ZIP هنا لو كانت البيئة تسمح (مباشرة داخل المحادثة قد لا يتم رفع ملفات تنفيذية خارجياً).
